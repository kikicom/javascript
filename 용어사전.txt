섀도잉(shadowing) : 내부 변수가 외부 변수를 가리는 현상
스코프(scope) : 식별자가 같은 단계에 있으는 경우 무조건 충동
메소드 체이닝 : 메소드가 리턴하는 값을 기반으로 해서 함수를 줄줄이 사용하는 것
트랜스파일러 : Babel등 최신 버전의 자바스크립트를 구 버전의 자바스크립트로 변경해주는
엄격모드(strict mode) : 여러 자바스크립트 코드를 보면 블록의 가장 위쪽에 'use strict'라는 문자열이 등장하는데 
                        엄격모드라고 부르는 기능으로 자바스크립트는 이러한 문자열을 읽어들인 순간부터 코드를 엄격하게 검사합니다. 
/*
'use strict'
data = 100
선언 오류 발생
*/
